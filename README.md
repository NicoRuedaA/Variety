# Ejercicios de Programación

Este repositorio contiene una colección de ejercicios básicos organizados por lenguaje de programación y tecnología.

## Estructura del Repositorio

- **Business_Central_AL**: Ejercicios relacionados con Microsoft Dynamics 365 Business Central (lenguaje AL).
- **C++**: Ejercicios básicos de programación en C++.
- **CSharp**: Ejercicios básicos de programación en C# (.NET).
- **Git_Practice**: Ejercicios y pruebas para practicar el uso de Git.
- **HTML**: Ejercicios básicos de maquetación web con HTML5.
- **Java**: Ejercicios de introducción a la programación con Java.
- **Python**: Ejercicios de introducción a la programación con Python.
- **SQL**: Scripts y ejercicios de bases de datos (Oracle SQL).

## Objetivo
El objetivo de este repositorio es consolidar el aprendizaje de diferentes bases de la programación en un solo lugar.
